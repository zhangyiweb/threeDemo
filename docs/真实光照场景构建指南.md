## 快速要点

1. **渲染器**：`SRGBColorSpace` + `ACESFilmicToneMapping` + `toneMappingExposure`
2. **IBL**：`PMREMGenerator` + `scene.environment`（Sky 烘焙 / HDR / RoomEnvironment）
3. **主光**：`DirectionalLight` + 阴影正交相机框罩住场景
4. **补光**：`HemisphereLight` 填暗部
5. **材质**：`MeshStandardMaterial`（metalness / roughness）
6. **昼夜**：`updateSun(hour)` 统一驱动天空、灯光、雾、曝光、云

详细章节请在本地打开仓库中的完整版 `docs/真实光照场景构建指南.md` 或查看 `zhtml/11.真实光照.html` 末尾注释。
